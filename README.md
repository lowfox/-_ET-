## 🐳BUILD
以下のコマンドを実行すると、EV3で動作するバイナリが`releases`ディレクトリに`app`という名前で生成されます。
```
$ make build
```

生成されるバイナリの名前は変更可能です。たとえば`ryuji`という名前にしたい場合、以下のように環境変数`APP_NAME`を指定してください。
```
$ make build APP_NAME=ryuji
```
